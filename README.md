# 📌 Sistema de Gerenciamento de Usuários
🧩 Sobre o Projeto

Este software foi desenvolvido com o objetivo de oferecer uma solução simples, eficiente e segura para o gerenciamento de usuários.

A empresa criada é completamente fictícia. 

A aplicação permite o cadastro, autenticação (login) e visualização de informações de forma prática e organizada, proporcionando uma experiência fluida para o usuário.

💻 Tecnologias Utilizadas
Java
Java Swing (GUI)
Banco de Dados SQL
🎯 Funcionalidades

Entre as principais funcionalidades do sistema, destacam-se:

✅ Cadastro de novos usuários com validação de dados
🔐 Sistema de login com verificação de credenciais
📋 Exibição das informações cadastradas de forma clara e estruturada
🖥️ Interface

O sistema foi construído utilizando formulários gráficos (GUI), oferecendo uma interface intuitiva e de fácil utilização, mesmo para usuários sem conhecimento técnico avançado.

🗄️ Banco de Dados

Para o armazenamento e gerenciamento das informações, é utilizado um banco de dados SQL, garantindo:

Integridade dos dados
Confiabilidade
Rapidez no acesso às informações
🚀 Objetivo do Projeto

O projeto foi pensado para servir como uma base sólida para sistemas maiores, podendo ser facilmente expandido com novas funcionalidades conforme a necessidade.

🧠 Considerações Finais

Este software demonstra a integração entre:

Interface gráfica
Lógica de programação
Banco de dados

Resultando em uma aplicação funcional, organizada e de fácil manutenção.
